# **Personal Portfolio Website | muhammadhassaanjaved.com**

This repository contains the complete source code and assets for my personal portfolio website, live at [**muhammadhassaanjaved.com**](https://muhammadhassaanjaved.com).

This site showcases my projects, publications, and a collection of useful web tools I've built. It is designed to be a clean, responsive, and interactive representation of my skills and work.

---

## ✨ **Key Features**

* **Responsive Design:** Fully functional and aesthetic on all devices, from mobile phones to desktops.
* **Dynamic Style Switcher:** Allows visitors to choose from multiple color themes to personalize their viewing experience.
* **Interactive Portfolio:** A gallery showcasing my various projects with images and descriptions.
* **Web Tools Hub:** A dedicated page (`tools.html`) that acts as a central hub for a suite of single-purpose utility applications.
* **Contact Form Integration:** Includes a functional contact form powered by EmailJS.
* **CI/CD Pipeline:** Automatically deploys to GitHub Pages on pushes to the `main` branch using GitHub Actions.

## 🛠 **Tech Stack**

This website is built with a focus on simplicity and performance using fundamental web technologies:

* **HTML5**
* **CSS3** (with custom properties for theming)
* **Vanilla JavaScript** (for interactivity and DOM manipulation)
* **Tailwind CSS** (used specifically for the `tools.html` hub)
* **GitHub Actions** (for automated deployment)

## 📂 **Repository Structure**

The repository is organized to be straightforward and easy to navigate:

```bash
.
├── index.html              # Main portfolio landing page
├── tools.html              # The web tools hub/landing page
├── cv.pdf                  # My downloadable resume
├── css/
│   ├── style.css           # Main stylesheet
│   └── skins/              # Different color theme stylesheets
├── js/
│   ├── script.js           # Core interactivity logic
│   └── style-switcher.js   # Logic for changing color themes
├── images/                 # Contains all images for the portfolio and publications
└── .github/workflows/
    └── deploy.yml          # GitHub Actions workflow for deployment
```

## 💻 **Local Development**

To run this project on your local machine:

1. **Clone the repository:**

   ```bash
   git clone https://github.com/itxcrusher/itxcrusher.github.io.git
   ```

2. **Navigate to the directory:**

   ```bash
   cd itxcrusher.github.io
   ```

3. **Open the project:**
   Simply open `index.html` in your favorite browser.
   *No special servers or dependencies are required.*
