# Muhammad Hassaan Javed | Site

Canonical personal site for Muhammad Hassaan Javed.

This repository is the public trust anchor for the broader ecosystem:

- personal identity and profile
- selected work and writing
- links into portfolio, tools, and framework surfaces

This repo is not the portfolio and it is not the tools hub.

## Stack

- Astro
- Crusher UI Kit
- GitHub Pages

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Deployment

The site is deployed through GitHub Pages and intended to serve the custom domain:

- `muhammadhassaanjaved.com`

Build output is static and deploys from `dist/`.
